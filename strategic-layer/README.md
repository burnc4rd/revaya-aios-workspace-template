# Strategic Layer

> The brain of the AI OS. Every decision, task, and initiative is filtered through one lens — the single most important goal.

---

## What This Is

The Strategic Layer is not background reading. It is an active decision filter that Claude loads before any prioritization, planning, or routing task. Every component of this AIOS is infused with the strategic context defined here.

**The four files in this layer:**

| File | Purpose | When Claude loads it |
|------|---------|---------------------|
| `oobg.md` | One Objective, One Bottleneck, One Goal — the core filter | Every `/build-guide`, `/reflect`, `/review`, `/metrics`, `/process` run |
| `unique-vehicle.md` | Your differentiation — why someone picks your business | Any content, sales, or positioning work |
| `priorities.md` | Current ranked priorities across all active work | Every `/build-guide` run |
| `direction.md` | Long-term horizon — what winning looks like | Strategic decisions, `/board`, monthly/quarterly reviews |

---

## How Claude Uses This Layer

**Before prioritizing:** Load `oobg.md`. Score every candidate task against the One Objective. Flag anything that doesn't connect to the current Bottleneck.

**Before content or sales work:** Load `unique-vehicle.md`. Every claim, hook, and angle must connect to the differentiator.

**Before strategic decisions:** Load `direction.md`. Does this choice move toward the horizon or away from it?

**The filter question:** "Does this task, decision, or initiative directly serve the One Objective or address the current Bottleneck? If not — kill it, park it, or delegate it."

---

## How to Update This Layer

- **OOBG** — review monthly (Monthly Strategic Reset cadence in Learning Loops). The Bottleneck shifts as obstacles are cleared.
- **Unique Vehicle** — stable. Update only if positioning fundamentally changes.
- **Priorities** — update weekly during Learning Loops Weekly Synthesis (`/review`).
- **Direction** — update quarterly. The horizon moves as wins accumulate.

---

## Connection to the Cycle

Strategic Layer → feeds → Prioritization Engine (`/build-guide`)
Learning Loops → refines → Strategic Layer (via Monthly Strategic Reset)
Knowledge Management → feeds → Strategic Layer (refined contextual knowledge)
